# Lab 4 - Debugging Challenge

## Goal

Experience how verification engineers debug failing tests.

## Scenario

A bug has been introduced into the CPU simulator.

The ADD instruction may not behave correctly.

## Your Job

1. Run the testbench
2. Identify which test fails
3. Find the wrong behaviour
4. Explain the root cause
5. Fix the code
6. Run the testbench again

## Hint

Compare the expected value of x3 in the arithmetic test with the actual value.

## Submission

Submit:

`debugging_report.pdf`

Your report should include:

- symptom
- investigation
- root cause
- fix
- final test result
