# Lab 2 - Run Your First CPU Simulation

## Goal

Compile and run the RISC-V testbench.

## Step 1: Open Terminal

Go to the starter folder:

```bash
cd starter
```

## Step 2: Build

```bash
make
```

## Step 3: Run

```bash
./riscv_testbench
```

## Step 4: Record Results

Copy the terminal output into your lab notes.
