# Student Assessment Rubric

| Category | Excellent | Good | Developing |
|---|---|---|---|
| Participation | Active and consistent | Usually engaged | Limited engagement |
| Technical Understanding | Explains ideas clearly | Understands basics | Needs support |
| Verification Tests | Multiple clear tests | Basic tests complete | Incomplete tests |
| Debugging | Evidence-based reasoning | Some reasoning | Mostly guessing |
| Report | Clear and structured | Understandable | Missing detail |
| Presentation | Confident and clear | Mostly clear | Hard to follow |

## Recommendation Letter Consideration

Recommendation letters are not automatic.

Students may be considered if they demonstrate:

- strong participation
- meaningful technical contribution
- clear project understanding
- professional communication
- strong final deliverables

Template letters are not used.
