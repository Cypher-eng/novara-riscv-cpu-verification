# Week 1 Lecture - CPU Foundations

## Big Question
How does a computer understand and execute instructions?

## Key Ideas
- A CPU executes instructions step by step.
- Registers store small values inside the CPU.
- Memory stores data and instructions.
- An ISA defines the language a CPU understands.

## Core Vocabulary
- CPU
- Instruction
- Register
- Memory
- ISA
- RISC-V

## Simple Example
`ADDI x1, x0, 5`

This means:

- take the value in register x0
- add 5
- store the result in register x1

Since x0 is always 0, x1 becomes 5.
