# Week 3 Lecture - Verification Engineering

## Big Question
How do engineers know a processor works correctly?

## Why Verification Matters
Modern chips are too complex to check manually. Verification engineers write tests to catch bugs before hardware is manufactured.

## Types of Tests
- Arithmetic tests
- Memory tests
- Branch tests
- Edge case tests

## Good Test Design
A good test should have:

- clear purpose
- expected output
- pass/fail result
